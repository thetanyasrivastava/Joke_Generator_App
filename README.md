# 😂 Joke Generator App

## Features
- Fetch random jokes using API
- Programming jokes support
- Multiple jokes rendering (loop)
- Built with Express + Axios + EJS

## Tech Stack
- Node.js
- Express
- Axios
- EJS

## How to Run
npm install
node index.js

## What I Learned
- API integration using Axios
- Handling async/await
- Error Handling
- Working with different API response formats (object vs array)
- Rendering dynamic content using EJS

## API Used
Official Joke API
